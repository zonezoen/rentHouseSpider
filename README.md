## 租房爬虫
- python 爬虫
- 数据分析
- 数据可视化

本项目爬取的是深圳的租房信息

#### 爬虫技术分析
- 请求库：requests
- HTML 解析：BeautifulSoup
- 词云：wordcloud
- 数据可视化：pyecharts
- 数据库：MongoDB
- 数据库连接：pymongo
